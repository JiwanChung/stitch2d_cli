stitch2D-CLI
========

Python Library to stitch microscopic images.


Install
-------

Install with pip:

```bash
pip install https://github.com/JiwanChung/stitch2d_cli.git
```

Quick start
-----------

```bash
stitch2d-cli ./examples/inputs2/0.png ./examples/inputs2/1.png -o ./examples/output2.png
```
